# Meta Ad Performance Dashboard

META AD PERFORMANCE DASHBOARD

It solves the problem of optimizing marketing spend and maximizing the return on investment (ROI) for Meta ad campaigns
by providing a unied, interactive dashboard for data-driven campaign management

This dashboard was built using the following tools and technologies:

-- Power Bi Desktop - Main data visualization platform used for report creation.

-- Power Query - Data transformation and cleaning layer for reshaping and preparing the data.

-- DAX (Data Analysis Expressions) - Used for calculated measures, dynamic visulas, and conditional logic.

-- Data Modeling - Relationships established among tables (ads, ad_Events, Users, Dynamic Measures and Calender table) to enable cross-filtering and aggregation.

-- File Format - .pbix for development and .png for dashboard previews.

Data Source - Data was imported from various sites including Meta and Kaggle.

Key Features:-

Key Metrics Tracked: Developed a dynamic, interactive Power BI dashboard that visualizes key KPIs (ROAS, CPA, CTR)
and segment performance by audience and creative, reducing the manual reporting time for stakeholders by 75% (e.g.,
from 4 hours to 1 hour weekly).

Purpose of Analysis: Identied cost ineciencies within the top 5 highest-spending ad sets by visualizing Cost Per Click
(CPC) trends, providing stakeholders with clear insights that contributed to a potential 5-10% optimization of the overall
marketing budget.

Decision-Making Tool: Enabled data-driven decisions on budget reallocation and creative testing by surfacing real-time
performance metrics, facilitating a shift of focus toward strategies with a 20% higher Return on Ad Spend (ROAS).

Technology: The project uses Microsoft Power BI to create a dynamic, interactive dashboard that makes complex data
accessible and understandable.
